FIFO (First-In-First-Out)

FIFO is the simplest page replacement algorithm that replaces the oldest page in memory. It is easy to implement but can lead to suboptimal performance, especially when the oldest page is still frequently used.

LRU (Least Recently Used)

LRU keeps track of the most recently used pages and replaces the one that hasn't been used for the longest time. It improves performance over FIFO but requires additional tracking of page usage.

Optimal

The Optimal page replacement algorithm replaces the page that will not be used for the longest time in the future. While this provides the best possible performance, it is not practical in real systems as it requires knowledge of future page requests.
